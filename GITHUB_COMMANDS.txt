========================================
GITHUB UPLOAD - QUICK COMMANDS
========================================

STEP-BY-STEP COMMANDS:
======================

1. OPEN TERMINAL IN PROJECT FOLDER
   (Right-click folder → Open in Terminal)

2. INITIALIZE GIT (if not done)
   git init

3. ADD REMOTE REPOSITORY
   git remote add origin https://github.com/YOUR_USERNAME/Attendance-Managment-System-Using-Face-Recognition-Yolo-DeepFace.git

4. CHECK STATUS
   git status

5. ADD ALL FILES
   git add .

6. COMMIT
   git commit -m "Initial commit: Face Recognition Attendance System"

7. SET BRANCH NAME
   git branch -M main

8. PULL EXISTING (if any)
   git pull origin main --allow-unrelated-histories

9. PUSH TO GITHUB
   git push -u origin main

TROUBLESHOOTING:
================

Error: "remote origin already exists"
→ git remote remove origin
→ Then add again

Error: "failed to push"
→ git pull origin main --rebase
→ git push origin main

Error: "authentication failed"
→ Use Personal Access Token as password
→ GitHub Settings → Developer settings → Tokens
